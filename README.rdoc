10 Mile Stereo

This application is for music lovers everywhere. A microblog where users can post music reviews, music news, even pictures. Even follow other users that post cool stuff.
This application was built on Ruby on Rails, with javascript, HTML, CSS and deployed to Heroku. Amazon Web services was used for user picture upload data and Heroku sendgrid addon to send activation and password reset emails. 
Users must have an account to view site, along with a valid email address to activate account login.
